公开进度摘要

用一页看懂 Roadmap 顺序和 implementation 状态。

Mycel 目前仍处在窄版 first-client lane。眼前目标是先收掉 shared-core 的 parsing、 canonicalization、replay 和 rebuild，再向 reader、sync 和 app-layer 扩展。

  1. 收掉 M1 的 malformed-field depth 和 canonicalization debt。
  2. 完成 M2 的 replay、rebuild 和 narrow write-path closure。
  3. 在 shared core 稳定之前，延后 M3+ 的扩展。

Milestone Timeline

这份 roadmap 是串行推进的：后面的 phase 都默认前面的 phase 已经稳定。现在的重心仍然是 M1 的收尾和 M2 的 closure。

Implementation Matrix

这张表把 checklist 压缩成 section-level 状态，让我们快速看到哪些部分已经大致完成、哪些仍然是 partial、哪些仍属于后续 phases。

Area Status Milestone Current read
1. Repo and Build Setup Mostly done M1 只剩 shared canonical JSON utility 还没有收尾。
2. Object Types and IDs Partial M1 所有必需 v0.1 families 都已经 typed；剩余工作是 malformed field-shape depth、semantic-edge closure 和角色建模。
3. Canonical Serialization and Hashing Partial M1 核心规则和 reproducibility coverage 已经存在；state_hash 和 wire 仍然缺 shared canonical utility reuse。
4. Signature Verification Partial M1 Object signature rules 大多已到位，且 signature-edge verify smoke coverage 更完整；wire-envelope verification 还没有开始。
5. Patch and Revision Engine Mostly done M2 Replay 和 state_hash 已经存在；patch-op base 已经很稳。
6. Local State and Storage Mostly done M2 Store ingest、rebuild 和 indexes 已经存在；local transport/safety separation 还没有完成。
7. Wire Protocol Not started M4 Canonical wire envelope 和 message validation 仍然属于后续工作。
8. Sync Workflow Not started M4 First-time 和 incremental sync 都还属于后续工作。
9. Views and Head Selection Mostly done M3 Deterministic selector core 已经存在；multi-profile 和 dual-role closure 还没有完成。
10. Merge Generation Partial M2 Verification 采用 replay-based，但 local merge authoring 还没有建立。
11. CLI or API Surface Partial M2 / M3 Verification 和 inspection 已经存在；authoring、sync 和 workflow split 还没有完成。
12. Interop Test Minimum Partial M1 / M2 Fixture isolation、reproducibility 和 smoke coverage 已经存在，但多条 normative wire 和 replay checks 还没有完成。
13. Ready-to-Build Gate Partial Whole plan Replay、head selection 和 rebuild 已经绿了；parse、wire sync 和 merge generation 还没有完成。
