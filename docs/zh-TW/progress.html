公開進度摘要

用一頁看懂 Roadmap 順序與 implementation 狀態。

Mycel 目前仍在窄版 first-client lane。眼前目標是先收掉 shared-core 的 parsing、canonicalization、 replay 與 rebuild，同時只透過具備 editor-admission 感知的 accepted-head inspect/render 與 profile workflows 謹慎擴張 reader surface。

  1. 收掉 M1 的 malformed-field depth 與 canonicalization debt。
  2. 完成 M2 的 replay、rebuild 與 merge-authoring closure。
  3. 在 accepted-head inspect/render 與具名 profile 基礎上，謹慎擴張 M3 reader workflows。

Implementation Matrix

這張表把 checklist 壓成 section-level 狀態，讓我們快速看到哪些區塊已大致完成、哪些仍屬 partial、哪些仍在後續 phases。

Area Status Milestone Current read
1. Repo and Build Setup Mostly done M1 build/test 基礎已穩定，shared canonical helper ownership 也更集中；完整 utility closure 仍未收尾。
2. Object Types and IDs Partial M1 所有必要 v0.1 families 都已 typed，且 top-level core-version 與 nested-path strictness 更完整；剩餘工作是 malformed field-shape depth、剩餘 semantic-edge closure 與角色建模。
3. Canonical Serialization and Hashing Partial M1 核心規則與 reproducibility coverage 已存在，且 canonical helper reuse 已更集中；`state_hash` 與 wire 仍缺完整 shared utility reuse。
4. Signature Verification Partial M1 Object signature rules 大多已到位，且 signature-edge verify smoke coverage 更完整；wire-envelope verification 尚未開始。
5. Patch and Revision Engine Mostly done M2 Replay 與 state_hash 已存在；對引用依賴的驗證與 sibling declared-ID determinism 也更嚴格。
6. Local State and Storage Mostly done M2 Store ingest、rebuild 與 indexes 已存在；local transport/safety separation 尚未完成。
7. Wire Protocol Not started M4 Canonical wire envelope 與 message validation 仍屬未來工作。
8. Sync Workflow Not started M4 First-time 與 incremental sync 都仍屬未來工作。
9. Views and Head Selection Mostly done M3 Deterministic selector core、具名 fixed-profile selection，以及具備 editor-admission 感知的 inspect/render flows 已存在；dual-role closure 尚未完成。
10. Merge Generation Partial M2 Replay-based verification 與保守型 local merge authoring 已存在，涵蓋 structural move/reorder、新 parent reparenting、簡單 composed parent-chain coverage，以及更廣的 nested structural matrix，但更豐富的 nested/reparenting conflict cases 仍會落回 manual curation。
11. CLI or API Surface Partial M2 / M3 Verification、authoring、保守型 merge authoring、具備 editor-admission 感知的 reader inspection/render、governance inspect/list/publish，以及 persisted governance index query surfaces 已存在；sync 仍未完成。
12. Interop Test Minimum Partial M1 / M2 Fixture isolation、reproducibility、更嚴格的 parser/replay smoke coverage，以及更廣的 merge-authoring structural matrix 已存在，但多條 normative wire 與 replay checks 尚未完成。
13. Ready-to-Build Gate Partial Whole plan Replay、head selection、rebuild 與保守型 merge authoring 已綠；parse closure 與 wire sync 尚未完成。
